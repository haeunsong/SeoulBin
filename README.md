## 1. 프로젝트 개요
- 프로젝트 이름 : 서울빈(SeoulBin)
- 목적 : 서울시 쓰레기통 위치 시각화 및 관리 서비스 구현

## 2. 사용한 기술 스택
- 프론트엔드 : Java Swing, JxBrowser, JavaScript
- 백엔드 : Java, Sqlite3
- 인공지능 : Roboflow 를 활용하여 쓰레기통 이미지 분류 모델 생성
- 외부 서비스 통합 : Kakao Maps API
